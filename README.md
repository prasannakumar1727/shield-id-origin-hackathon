# Shield-ID: Agentic AI Identity Firewall

## ORIGIN – National AI Hackathon (SIMATS Engineering)

---

## 🚨 Problem Statement

With the rise of generative AI, cybercriminals can now create hyper-realistic deepfake faces, synthetic identities, and AI-manipulated documents that bypass traditional KYC systems.

Financial institutions face:

- Increasing synthetic identity fraud
- Massive financial losses
- Regulatory risk
- Loss of digital trust

Current onboarding systems rely on static document checks and basic face matching, which are no longer sufficient in the AI era.

There is a need for an intelligent, real-time identity verification firewall.

---

## 💡 Our Solution

Shield-ID is an AI-powered identity verification system that prevents synthetic identity fraud during digital onboarding.

It performs multi-layer verification using:

- Deepfake detection
- Facial biometric comparison
- Real-time liveness detection
- ID document forgery analysis
- Behavioral anomaly detection

All signals are processed through an Agentic Risk Engine that generates a real-time Trust Score (0–100) with explainable fraud indicators.

Shield-ID verifies human authenticity before trust is granted.
